# Hotel Reservation & Analytics System

A MySQL-based Hotel Reservation & Analytics System designed to manage hotel operations efficiently. This project demonstrates database design, reservation management, customer management, payment processing, and business analytics using SQL.

## Project Overview

The Hotel Reservation & Analytics System is a relational database project developed using MySQL. It enables hotels to manage reservations, rooms, customers, employees, payments, invoices, housekeeping, services, discounts, and customer feedback. The project also includes analytical SQL queries to generate meaningful business insights.
